# AIWEBX — Siber Zorbalık Tespit ve Önleme Sistemi 🛡️

AIWEBX, siber zorbalığın tespiti ve önlenmesini amaçlayan yapay zekâ destekli bir web uygulamasıdır.
Ünides Projesi kapsamında düzenlenen 24 saatlik Hackathon'da **510BT** ekibi tarafından geliştirilmiş
ve **8 ekip arasından 3.lük ödülü** kazanmıştır.

---

## 🚀 Özellikler

- Metin tabanlı siber zorbalık tespiti (NLP)
- Yapay zekâ destekli içerik analizi
- Kullanıcı dostu web arayüzü

---

## 🛠️ Kullanılan Teknolojiler

- **Python** — Model geliştirme ve backend
- **Machine Learning / NLP** — Metin sınıflandırma
- **JavaScript / HTML / CSS** — Web arayüzü
